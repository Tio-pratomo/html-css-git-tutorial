## Materi: Pengetahuan & Konsep

Tujuan sesi ini: kamu benar-benar paham **cara pakai `columns` / `column` untuk layout dasar**, bukan sekadar menyalin contoh.

**Bulma punya dua “layer” layout:**

- **Columns**: layout 1 dimensi (baris ↔ kolom) berbasis Flexbox, cocok untuk membagi area horizontal.
- **Grid**: layout 2 dimensi (baris dan kolom) yang lebih baru (Smart Grid / Fixed Grid), kita bahas di sesi terpisah.

Di sesi 11 ini fokus di **Columns basics + sizes**.

### 1. Konsep dasar: `columns` & `column`

Aturan main columns Bulma:

- Bungkus baris dengan `<div class="columns">`.
- Di dalamnya, buat satu atau lebih `<div class="column">`.
- Secara default, **semua `column` akan punya lebar sama**, berapapun jumlahnya.

**Contoh paling dasar:**

```html
<div class="columns">
  <div class="column">Kolom 1</div>
  <div class="column">Kolom 2</div>
  <div class="column">Kolom 3</div>
</div>
```

- Ini akan membagi lebar jadi 3 bagian sama besar (1/3, 1/3, 1/3).
- Kalau 4 `column`, otomatis 1/4 masing-masing; kalau 2, jadi 1/2 masing-masing.

**Hints:**

- `columns` adalah flex container (`display: flex`) dengan beberapa pengaturan default (wrap, gap, dll.).
- `column` adalah flex item yang grow/shrink otomatis.

### 2. Sistem ukuran: 12-grid, fractional, dan numeric

Kalau semua auto sama besar tidak cukup, kamu bisa mengatur lebar masing-masing `column` dengan **modifier ukuran**.

Ada dua gaya:

1. **Fractional helpers** (lebih “manusiawi”):
   - `is-two-thirds` → 8/12 lebar.
   - `is-three-quarters` → 9/12.
   - `is-half` → 6/12.
   - `is-one-third` → 4/12.
   - `is-one-quarter` → 3/12, dll.

2. **Numeric (12-grid)**:
   - `is-1` sampai `is-12` → 1/12 sampai 12/12 lebar.
   - Contoh: `is-4` = 4/12 (≈ 33%), `is-8` = 8/12 (≈ 67%).

Contoh kombinasi numeric:

```html
<div class="columns">
  <div class="column is-4">4/12 (33%)</div>
  <div class="column is-8">8/12 (67%)</div>
</div>
```

Contoh kombinasi fractional:

```html
<div class="columns">
  <div class="column is-one-quarter">1/4</div>
  <div class="column is-half">1/2</div>
  <div class="column">Sisa (auto)</div>
</div>
```

**Di contoh kedua:**

- Kolom 1: 3/12.
- Kolom 2: 6/12.
- Kolom 3: sisa (3/12).

Ini persis seperti yang sudah muncul sekilas di materi sesi 1, sekarang kita formalkan sebagai “aturan main ukuran columns”.

### 3. Satu kolom fixed + lainnya auto

Kombinasi paling sering di dashboard: satu kolom fixed (misalnya sidebar kecil atau panel filter), sisa lebar dipakai oleh kolom lain.

Contoh:

```html
<div class="columns">
  <div class="column is-3">Filter panel (3/12)</div>
  <div class="column">Main content (auto, sisa)</div>
</div>
```

Karena kolom kedua tidak diberi ukuran, dia akan mengambil **sisa lebar** setelah kolom pertama.

Contoh lain:

```html
<div class="columns">
  <div class="column is-one-third">Sidebar (1/3)</div>
  <div class="column">Konten utama</div>
</div>
```

Pola ini sangat berguna untuk layout 2 kolom sederhana (sidebar + konten).

### 4. Multi-row dengan `is-multiline` (sekilas)

Kalau jumlah `column` melebihi lebar baris (terutama di mobile), kamu bisa mengizinkan wrap dengan `is-multiline` di `columns`.

```html
<div class="columns is-multiline">
  <div class="column is-half">Item 1</div>
  <div class="column is-half">Item 2</div>
  <div class="column is-half">Item 3</div>
  <div class="column is-half">Item 4</div>
</div>
```

- Di layar lebar: bisa saja muat 4 per baris (tergantung container).
- Di layar sempit: kolom-kolom akan pindah ke baris berikutnya otomatis (wrap).

Responsiveness per breakpoint (`is-half-tablet`, `is-one-third-desktop`, dst.) akan kita bahas fokus di sesi khusus Columns Responsiveness.

## Praktik

Di Sesi 11 ini kita bikin **halaman demo khusus columns** supaya kamu bisa eksplor kombinasi ukuran tanpa ganggu halaman lain.

### Langkah 1: Buat `columns-demo.html`

**Buat file baru:**

```html
<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <title>Bulma Columns Demo</title>
    <link
      rel="stylesheet"
      href="https://cdn.jsdelivr.net/npm/bulma@1.0.4/css/bulma.min.css"
    />
  </head>
  <body>
    <section class="section">
      <div class="container">
        <h1 class="title">Columns Demo</h1>

        <!-- Area latihan di bawah ini -->
      </div>
    </section>
  </body>
</html>
```

Semua latihan berikut ditempatkan di dalam `<div class="container">`.

### Langkah 2: Equal columns (2, 3, 4 kolom)

Tambahkan blok:

```html
<h2 class="title is-4 mt-5">Equal Columns</h2>

<div class="columns">
  <div class="column has-background-primary-light">Kolom 1</div>
  <div class="column has-background-link-light">Kolom 2</div>
</div>

<div class="columns">
  <div class="column has-background-info-light">Kolom 1</div>
  <div class="column has-background-success-light">Kolom 2</div>
  <div class="column has-background-warning-light">Kolom 3</div>
</div>

<div class="columns">
  <div class="column has-background-danger-light">1</div>
  <div class="column has-background-primary-light">2</div>
  <div class="column has-background-link-light">3</div>
  <div class="column has-background-info-light">4</div>
</div>
```

Perhatikan bagaimana semua kolom membagi lebar sama rata tanpa kamu atur ukuran apa pun.

### Langkah 3: Numeric sizes (12-grid)

Tambahkan:

```html
<h2 class="title is-4 mt-5">Numeric Sizes (is-*)</h2>

<div class="columns">
  <div class="column is-4 has-background-primary-light">is-4 (4/12)</div>
  <div class="column is-8 has-background-link-light">is-8 (8/12)</div>
</div>

<div class="columns">
  <div class="column is-3 has-background-info-light">is-3 (3/12)</div>
  <div class="column is-3 has-background-success-light">is-3 (3/12)</div>
  <div class="column has-background-warning-light">auto (sisa)</div>
</div>
```

Coba modifikasi angka:

- Ubah `is-4` jadi `is-5`, lihat perubahan proporsi.
- Pastikan total tidak harus 12; Bulma tetap akan mengatur flex item dengan baik, tapi berpikir dalam 12-grid membantu konsistensi desain.

### Langkah 4: Fractional sizes

Tambahkan:

```html
<h2 class="title is-4 mt-5">Fractional Sizes</h2>

<div class="columns">
  <div class="column is-one-quarter has-background-primary-light">1/4</div>
  <div class="column is-half has-background-link-light">1/2</div>
  <div class="column has-background-info-light">auto (sisa)</div>
</div>

<div class="columns">
  <div class="column is-two-thirds has-background-success-light">2/3</div>
  <div class="column has-background-warning-light">auto (1/3)</div>
</div>
```

Latihan kecil:

- Ganti `is-one-quarter` jadi `is-one-third` dan lihat bagaimana layout berubah.
- Bandingkan penggunaan fractional (`is-one-third`) vs numeric (`is-4`), dan pilih mana yang lebih natural untuk kamu ingat.

### Langkah 5 (opsional): Multiline simple

Tambahkan:

```html
<h2 class="title is-4 mt-5">Multiline</h2>

<div class="columns is-multiline">
  <div class="column is-half has-background-primary-light">Item 1</div>
  <div class="column is-half has-background-link-light">Item 2</div>
  <div class="column is-half has-background-info-light">Item 3</div>
  <div class="column is-half has-background-success-light">Item 4</div>
</div>
```

Resize browser atau pakai DevTools mobile view dan lihat bagaimana kolom-kolom ini pindah baris saat viewport mengecil.

---

Kalau kamu sudah nyaman dengan:

- `columns` vs `column`
- Auto width vs `is-*` numeric/fractional
- Kombinasi “1 fixed + 1 auto”

berarti fondasi columns kamu sudah lumayan kuat.

Di **Sesi 12**, kita bisa fokus ke:

- **Columns responsiveness** per breakpoint (`is-half-tablet`, `is-one-third-desktop`, dll.).
- **Nesting**, `gap`, dan opsi alignment (`is-vcentered`, `is-variable is-4` untuk mengatur jarak antar kolom).

Kalau ada bagian dari latihan di atas yang kerasa “nggak klik” (misalnya: apa bedanya `is-4` vs `is-one-third`), bilang saja, nanti saya luruskan dulu sebelum lanjut.
