## Materi: Pengetahuan & Konsep

### 1. Table component Bulma

Tabel Bulma sebenarnya hanyalah `<table>` HTML biasa yang diberi kelas `table` dan beberapa modifier. Ini membuatmu tetap memakai struktur HTML standar sambil mendapat styling modern.

Modifier penting:

- `is-bordered`: menambah border di semua sel.
- `is-striped`: memberi warna selang-seling (zebra) agar baris mudah dibaca.
- `is-narrow`: mengurangi padding, cocok untuk data yang sangat banyak kolom.
- `is-hoverable`: highlight baris saat mouse melintas.
- `is-fullwidth`: membuat tabel mengisi lebar container 100%.

Contoh dasar:

```html
<table class="table is-striped is-hoverable is-fullwidth">
  <thead>
    <tr>
      <th>ID</th>
      <th>Title</th>
      <th>Status</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <th>1</th>
      <td>Mastering Bulma</td>
      <td>Published</td>
    </tr>
  </tbody>
</table>
```

**Responsiveness (wajib paham)**

Secara default tabel tidak responsif. Kalau kolom banyak, di HP bisa “meledak” ke luar layar. Solusi standar Bulma adalah membungkus tabel dengan `table-container`:

```html
<div class="table-container">
  <table class="table is-fullwidth">
    <!-- isi tabel -->
  </table>
</div>
```

Wrapper ini akan membuat scrollbar horizontal otomatis jika tabel terlalu lebar untuk layar kecil, tanpa merusak layout di luar tabel.

### 2. Dropdown component (bukan `<select>`)

Dropdown Bulma adalah komponen UI custom, beda dengan `<select>` HTML native. Isinya bisa link, teks biasa, divider, dsb., dan butuh sedikit JS jika ingin buka/tutup dengan klik (kecuali pakai `is-hoverable`).

Struktur inti:

```html
<div class="dropdown is-active">
  <!-- Trigger -->
  <div class="dropdown-trigger">
    <button class="button" aria-haspopup="true" aria-controls="dropdown-menu">
      <span>Dropdown button</span>
      <span class="icon is-small">
        <i class="fas fa-angle-down" aria-hidden="true"></i>
      </span>
    </button>
  </div>

  <!-- Menu -->
  <div class="dropdown-menu" id="dropdown-menu" role="menu">
    <div class="dropdown-content">
      <a href="#" class="dropdown-item">Link Item</a>
      <a class="dropdown-item is-active">Active Item</a>
      <hr class="dropdown-divider" />
      <div class="dropdown-item">Custom Text</div>
    </div>
  </div>
</div>
```

Kelas `is-active` di elemen `.dropdown` mengontrol apakah menu sedang tampil atau tidak.

Modifier penting:

- `is-right`: menu keluar ke kanan (berguna di tepi kanan halaman).
- `is-up`: menu muncul ke atas (dropup).
- `is-hoverable`: menu terbuka saat di-hover, jadi kamu bisa pakai tanpa JS untuk kasus sederhana.

## Praktik: `inventory.html` – Halaman Tabel Book Inventory

Kita buat halaman baru `inventory.html` untuk menampilkan data buku dalam bentuk tabel plus filter status memakai dropdown.

### Langkah 1: Setup `inventory.html`

1. Buat file `inventory.html`.
2. Copy struktur dasar dari `dashboard.html` (navbar, sidebar, layout 2 kolom, script burger).
3. Kosongkan area konten utama (kolom kanan), seperti di sesi sebelumnya saat membuat `add-book.html`.

Kita akan mengisi kolom kanan dengan toolbar + dropdown + tabel.

### Langkah 2: Toolbar dengan dropdown filter

Di dalam kolom kanan, paling atas:

```html
<nav class="level">
  <div class="level-left">
    <div class="level-item">
      <h1 class="title is-4">Book Inventory</h1>
    </div>
  </div>

  <div class="level-right">
    <div class="level-item">
      <!-- DROPDOWN FILTER -->
      <div class="dropdown is-hoverable is-right">
        <div class="dropdown-trigger">
          <button
            class="button"
            aria-haspopup="true"
            aria-controls="dropdown-menu4"
          >
            <span>Filter by Status</span>
            <span class="icon is-small">
              <i class="fas fa-angle-down" aria-hidden="true"></i>
            </span>
          </button>
        </div>
        <div class="dropdown-menu" id="dropdown-menu4" role="menu">
          <div class="dropdown-content">
            <div class="dropdown-item">
              <p>Select status to view:</p>
            </div>
            <hr class="dropdown-divider" />
            <a href="#" class="dropdown-item is-active">All Books</a>
            <a href="#" class="dropdown-item">Published</a>
            <a href="#" class="dropdown-item">Draft</a>
            <a href="#" class="dropdown-item">Archived</a>
          </div>
        </div>
      </div>
    </div>
  </div>
</nav>
```

- `dropdown is-hoverable is-right`: menu akan muncul saat di-hover, dan posisinya rata kanan terhadap tombol.
- `dropdown-item is-active`: status filter yang sedang dipilih secara visual (walau belum ada logika JS belakangnya).

### (Opsional) Langkah 4: Dropdown dengan klik, bukan hover

Jika kamu mau dropdown hanya terbuka saat diklik (bukan `is-hoverable`), lakukan:

1. Hapus `is-hoverable` di HTML (`<div class="dropdown is-right">`).
2. Tambahkan script ini sebelum `</body>`:

```html
<script>
  document.addEventListener("DOMContentLoaded", () => {
    const dropdowns = document.querySelectorAll(".dropdown:not(.is-hoverable)");

    if (dropdowns.length > 0) {
      dropdowns.forEach((el) => {
        el.addEventListener("click", (e) => {
          e.stopPropagation();
          el.classList.toggle("is-active");
        });
      });

      document.addEventListener("click", () => {
        dropdowns.forEach((el) => {
          el.classList.remove("is-active");
        });
      });
    }
  });
</script>
```

Logika: klik pada `.dropdown` akan toggle `is-active`; klik di luar akan menutup semua dropdown.
