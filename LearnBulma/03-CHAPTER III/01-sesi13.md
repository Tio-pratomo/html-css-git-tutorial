## Materi: Pengetahuan & Konsep

Di sesi 11–12 kita pakai **columns** (layout 1 dimensi: fokus baris ↔ kolom).  
Mulai Bulma v1 ada sistem **Grid 2D** baru dengan dua mode:

- **Smart Grid** – jumlah kolom fleksibel, ditentukan oleh **_minimum width_** tiap kolom.
- **Fixed Grid** – jumlah kolom fix (2, 3, 4, dst.), tidak peduli seberapa lebar kontennya.

**Keduanya pakai elemen dasar:**

```html
<div class="grid">
  <div class="cell">Cell 1</div>
  <div class="cell">Cell 2</div>
  ...
</div>
```

Setiap grid terdiri dari beberapa **cell** yang bisa kamu atur lebar/tingginya per cell, bahkan per breakpoint.

### 1. Smart Grid: minimum column width, auto-fit

Smart Grid adalah grid 2D **fleksibel**.

Bulma akan mengisi baris dengan sebanyak mungkin cell, **selama lebar tiap cell tidak kurang dari “minimum width”**.

Karakteristik utama Smart Grid:

- Default **min column width** = `9rem`.
- Default gap antar cell = `0.75rem`.
- Kalau viewport lebih lebar, akan muat lebih banyak cell per baris; kalau lebih sempit, cell otomatis turun ke baris berikutnya.

Contoh dasar Smart Grid:

```html
<div class="grid">
  <div class="cell has-background-primary-light">1</div>
  <div class="cell has-background-link-light">2</div>
  <div class="cell has-background-info-light">3</div>
  <div class="cell has-background-success-light">4</div>
  <div class="cell has-background-warning-light">5</div>
  <div class="cell has-background-danger-light">6</div>
</div>
```

**Efeknya:**

- Di layar lebar, mungkin semua 6 cell muat dalam dua baris.
- Di layar lebih kecil, jumlah cell per baris berkurang otomatis, tanpa kamu hitung manual `is-4`, `is-3`, dll.

Ini sangat enak untuk:

- Gallery foto / kartu produk.
- Daftar card “Latest Books” yang jumlahnya bisa berubah-ubah.

### 2. Fixed Grid: jumlah kolom fixed

Fixed Grid dipakai saat kamu mau **jumlah kolom yang pasti** (misal selalu 3 kolom), bukan based on minimum width.

Untuk itu, kamu bungkus grid di dalam container `fixed-grid` dan set jumlah kolom dengan modifier `has-$n-cols`:

```html
<div class="fixed-grid has-3-cols">
  <div class="grid">
    <div class="cell has-background-primary-light">1</div>
    <div class="cell has-background-link-light">2</div>
    <div class="cell has-background-info-light">3</div>
    <div class="cell has-background-success-light">4</div>
    <div class="cell has-background-warning-light">5</div>
    <div class="cell has-background-danger-light">6</div>
  </div>
</div>
```

- `has-3-cols` → grid dipecah jadi 3 kolom tetap; sisanya mengalir ke baris berikutnya.
- Tersedia dari `has-1-cols` sampai `has-12-cols`.

Yang menarik, kamu bisa atur **column count per breakpoint**:

Contoh:

```html
<div
  class="fixed-grid
         has-1-cols-mobile
         has-2-cols-tablet
         has-3-cols-desktop"
>
  <div class="grid">
    <div class="cell">1</div>
    <div class="cell">2</div>
    <div class="cell">3</div>
    <div class="cell">4</div>
    <div class="cell">5</div>
    <div class="cell">6</div>
  </div>
</div>
```

Artinya:

- Mobile: 1 kolom (stack vertikal).
- Tablet: 2 kolom.
- Desktop ke atas: 3 kolom.

Ada juga `has-auto-count`: fixed grid akan otomatis mengubah jumlah kolom per breakpoint, tapi untuk sekarang paham manual `has-n-cols-{breakpoint}` sudah cukup.

### 3. Grid Cells: mengatur lebar/tinggi per cell

Setiap `cell` di grid dapat dikontrol ukuran lebar (span) dan tinggi barisnya per breakpoint.

Konsepnya mirip CSS Grid `grid-column: span 2`, dll.

Contoh (skema umum, nama class bisa berbeda tergantung versi):

```html
<div class="grid">
  <div class="cell is-col-span-2 has-background-primary-light">
    Span 2 kolom
  </div>
  <div class="cell has-background-link-light">Normal</div>
  <div class="cell has-background-info-light">Normal</div>
</div>
```

- `is-col-span-2` → cell itu melebar menempati 2 kolom.
- Juga ada pattern serupa untuk tinggi (`is-row-span-2`), dan bisa per breakpoint (`is-col-span-2-desktop`, dsb.).

Pada praktik awal, kamu tidak wajib main span. Cukup pakai semua cell ukuran default dulu.

### 4. Kapan pakai Columns vs Grid?

Ringkasan pemakaian:

- Pakai **columns** bila:
  - Fokus pada 1 baris layout (sidebar + konten, 3 statistik, dsb.).
  - Layout lebih linier, tidak butuh cell berbeda-beda ukuran banyak.

- Pakai **Smart Grid** bila:
  - Mau menyusun banyak item dengan lebar minimum; jumlah kolom boleh fleksibel.
  - Contoh: gallery, daftar card yang berubah-ubah.

- Pakai **Fixed Grid** bila:
  - Mau kendali penuh jumlah kolom (2, 3, 4) dan per breakpoint.
  - Contoh: katalog dengan 3 kolom di desktop, 2 di tablet, 1 di mobile.

## Praktik

Kita buat file baru `grid-demo.html` sebagai playground grid 2D.

### Langkah 1: Setup `grid-demo.html`

```html
<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8" />
    <title>Bulma Grid Demo</title>
    <link
      rel="stylesheet"
      href="https://cdn.jsdelivr.net/npm/bulma@1.0.4/css/bulma.min.css"
    />
  </head>
  <body>
    <section class="section">
      <div class="container">
        <h1 class="title">Grid Demo</h1>

        <!-- Area latihan akan ditaruh di sini -->
      </div>
    </section>
  </body>
</html>
```

### Langkah 2: Smart Grid sederhana

Di dalam `container`:

```html
<h2 class="title is-4 mt-5">Smart Grid (min-width based)</h2>

<div class="grid">
  <div class="cell has-background-primary-light">1</div>
  <div class="cell has-background-link-light">2</div>
  <div class="cell has-background-info-light">3</div>
  <div class="cell has-background-success-light">4</div>
  <div class="cell has-background-warning-light">5</div>
  <div class="cell has-background-danger-light">6</div>
  <div class="cell has-background-primary-light">7</div>
  <div class="cell has-background-link-light">8</div>
  <div class="cell has-background-info-light">9</div>
</div>
```

Lalu:

- Resize lebar viewport dan amati bagaimana jumlah cell per baris berubah otomatis.

Kalau mau eksplor lebih jauh, dokumentasi menyebut kamu bisa mengatur min-width/gap lewat kelas utilitas dan custom CSS, tapi untuk pemula cukup paham perilaku default `9rem` + `0.75rem gap` dulu.

### Langkah 3: Fixed Grid dengan `has-n-cols`

Masih di `container`:

```html
<h2 class="title is-4 mt-5">Fixed Grid (has-n-cols)</h2>

<div class="fixed-grid has-3-cols">
  <div class="grid">
    <div class="cell has-background-primary-light">1</div>
    <div class="cell has-background-link-light">2</div>
    <div class="cell has-background-info-light">3</div>
    <div class="cell has-background-success-light">4</div>
    <div class="cell has-background-warning-light">5</div>
    <div class="cell has-background-danger-light">6</div>
  </div>
</div>
```

Coba:

- Ubah `has-3-cols` jadi `has-4-cols`, `has-2-cols`, dan lihat perubahan bentuk grid.

### Langkah 4: Fixed Grid responsif per breakpoint

Tambahkan:

```html
<h2 class="title is-4 mt-5">Fixed Grid Responsif</h2>

<div
  class="fixed-grid
         has-1-cols-mobile
         has-2-cols-tablet
         has-3-cols-desktop"
>
  <div class="grid">
    <div class="cell has-background-primary-light">1</div>
    <div class="cell has-background-link-light">2</div>
    <div class="cell has-background-info-light">3</div>
    <div class="cell has-background-success-light">4</div>
    <div class="cell has-background-warning-light">5</div>
    <div class="cell has-background-danger-light">6</div>
    <div class="cell has-background-primary-light">7</div>
    <div class="cell has-background-link-light">8</div>
    <div class="cell has-background-info-light">9</div>
  </div>
</div>
```

Gunakan DevTools:

- Di mobile width: cek bahwa hanya 1 kolom (items stack ke bawah).
- Di tablet: 2 kolom.
- Di desktop: 3 kolom.

### Langkah 5 (opsional): Main sedikit dengan cell span

Kalau versi Bulma kamu sudah mendukung helper span, coba satu contoh:

```html
<h2 class="title is-4 mt-5">Grid Cells (Span Contoh)</h2>

<div class="grid">
  <div class="cell has-background-primary-light">Normal</div>
  <div class="cell has-background-link-light">Normal</div>
  <div class="cell is-col-span-2 has-background-info-light">
    Span 2 kolom (cek dokumentasi Grid Cells untuk nama class yang tepat)
  </div>
  <div class="cell has-background-success-light">Normal</div>
</div>
```

Nama class span bisa berbeda sedikit tergantung versi Bulma, jadi sebaiknya cek halaman **Grid Cells** untuk daftar lengkap (lebar dan tinggi per cell).
