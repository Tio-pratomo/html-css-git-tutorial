title: Kustomisasi Tema Bulma dengan CSS Variables & Dark Mode Dasar
---

## Materi: Pengetahuan & Konsep

Mulai Bulma v1, hampir semua komponen distyling dengan **CSS variables** (`var(--bulma-... )`), bukan lagi nilai hard-coded. Ini membuat kustomisasi tema (warna, font, dark mode) jauh lebih mudah:

- Kamu **tidak wajib** pakai Sass untuk sekadar ganti warna utama.
- Cukup override CSS variables di `:root` atau pada elemen tertentu.

### 1. Dasar CSS variables Bulma

Contoh dari dokumentasi: elemen `.title` distyling seperti ini:

```css
.title {
  color: var(--bulma-title-color);
  font-size: var(--bulma-title-size);
  font-weight: var(--bulma-title-weight);
  line-height: var(--bulma-title-line-height);
}
```

Hal penting:

- Semua CSS variables Bulma **diprefix** `--bulma-...`. Contoh: `--bulma-primary-h`, `--bulma-primary-s`, `--bulma-primary-l` (hue, saturation, lightness warna primary).
- Variabel bisa didefinisikan:
  - di `:root` (global).
  - di scope komponen, misalnya `.button { --bulma-primary-h: ... }` untuk override spesifik.

Artinya: kamu bisa override tema global, atau bikin varian styling hanya di bagian tertentu.

### 2. Override tema dengan CSS biasa (tanpa Sass)

Cara paling simpel (recommended buat kamu sekarang):

1. Tetap load Bulma via CDN seperti biasa.
2. Tambahkan `<style>` setelah link Bulma, atau file CSS terpisah, lalu override variabel di `:root`.

Contoh: ubah warna utama (primary) dan warna background halaman:

```html
<head>
  ...
  <link
    rel="stylesheet"
    href="https://cdn.jsdelivr.net/npm/bulma@1.0.4/css/bulma.min.css"
  />

  <style>
    :root {
      /* Warna primary: hue/saturation/lightness */
      --bulma-primary-h: 210; /* biru keunguan */
      --bulma-primary-s: 100%;
      --bulma-primary-l: 45%;

      /* Background utama halaman (scheme main) */
      --bulma-scheme-main: hsl(210, 20%, 98%);
      --bulma-body-background-color: hsl(210, 20%, 96%);
    }
  </style>
</head>
```

Begitu kamu mengubah nilai ini, semua komponen yang memakai `is-primary` (button, navbar, link, tag, dsb.) otomatis pakai warna baru.

Praktisnya:

- Untuk eksperimen cepat, kamu bahkan bisa buka DevTools, cari `:root`, dan ubah nilai `--bulma-primary-*` langsung di browser untuk melihat efeknya real-time.

### 3. Scope kustomisasi: global vs lokal

Selain `:root`, kamu bisa override CSS variables di **scope yang lebih sempit**, misalnya layout tertentu.

Contoh: dashboard dengan tema primary khusus:

```html
<section class="section dashboard-theme">...</section>

<style>
  .dashboard-theme {
    --bulma-primary-h: 142; /* hijau */
    --bulma-primary-s: 80%;
    --bulma-primary-l: 40%;
  }
</style>
```

- Semua elemen Bulma **di dalam `.dashboard-theme`** akan menggunakan warna primary hijau ini, tanpa menyentuh halaman lain.
- Ini efektif untuk multi-section UI atau multi-brand dalam satu aplikasi.

### 4. Dark mode dasar dengan CSS variables

Bulma punya konsep dark mode yang memanfaatkan dua hal:

- Preferensi sistem: `@media (prefers-color-scheme: dark)`.
- Atribut atau class manual: `[data-theme="dark"]` atau `.theme-dark` yang mengubah CSS variables.

Pola umum di dokumentasi:

```css
@media (prefers-color-scheme: dark) {
  :root {
    /* Update CSS variables untuk dark scheme */
  }

  [data-theme="dark"],
  .theme-dark {
    /* Update CSS variables untuk memaksa dark theme */
  }
}
```

Konsepnya:

- Bulma membalik **lightness** scheme utama (background, border, text) untuk dark mode; hue & saturation tetap, tapi terang ↔ gelap dibalik.
- Dengan CSS variables, kamu cukup update sekumpulan variabel seperti `--bulma-scheme-main`, `--bulma-scheme-main-bis`, `--bulma-text`, dll., dan seluruh UI ikut berubah.

Untuk kamu sekarang, kita pakai versi yang lebih sederhana: **toggle class `.theme-dark` di `<html>` atau `<body>`** dan override beberapa variabel kunci.

## Praktik: Simple Theme & Dark Mode Toggle

Kita buat satu halaman demo, misalnya `theme-demo.html`, untuk:

- Mengganti warna primary default.
- Menambahkan tombol toggle Light/Dark yang mengubah class pada `<html>` / `<body>` dan beberapa CSS variables.

### Langkah 1: Setup `theme-demo.html`

```html
<!DOCTYPE html>
<html lang="id" class="">
  <head>
    <meta charset="UTF-8" />
    <title>Bulma Theme Demo</title>
    <link
      rel="stylesheet"
      href="https://cdn.jsdelivr.net/npm/bulma@1.0.4/css/bulma.min.css"
    />

    <style>
      /* Tema default (light) */
      :root {
        --bulma-primary-h: 222;
        --bulma-primary-s: 85%;
        --bulma-primary-l: 56%;
        --bulma-body-background-color: hsl(0, 0%, 98%);
        --bulma-scheme-main: hsl(0, 0%, 100%);
        --bulma-text: hsl(0, 0%, 14%);
      }

      /* Tema dark (di-scope ke .theme-dark) */
      .theme-dark {
        --bulma-scheme-main: hsl(210, 15%, 12%);
        --bulma-body-background-color: hsl(210, 15%, 10%);
        --bulma-text: hsl(0, 0%, 96%);
        /* sedikit atur warna primary agar tetap kontras di dark bg */
        --bulma-primary-l: 60%;
      }
    </style>
  </head>
  <body>
    <section class="section">
      <div class="container">
        <div
          class="is-flex is-justify-content-space-between is-align-items-center mb-5"
        >
          <h1 class="title">Bulma Theme & Dark Mode Demo</h1>
          <button id="theme-toggle" class="button is-primary">
            Toggle Dark Mode
          </button>
        </div>

        <div class="columns">
          <div class="column">
            <div class="box">
              <h2 class="title is-4">Primary Button</h2>
              <button class="button is-primary">Primary</button>
              <button class="button is-primary is-outlined">Outlined</button>
            </div>
          </div>
          <div class="column">
            <div class="box">
              <h2 class="title is-4">Notification</h2>
              <div class="notification is-primary">
                <button class="delete"></button>
                Ini notifikasi dengan warna primary.
              </div>
            </div>
          </div>
        </div>
      </div>
    </section>

    <script>
      const toggleBtn = document.getElementById("theme-toggle");
      const html = document.documentElement; // <html>

      toggleBtn.addEventListener("click", () => {
        html.classList.toggle("theme-dark");
      });
    </script>
  </body>
</html>
```

Yang terjadi di sini:

- Default: tidak ada `theme-dark` di `<html>` → pakai variabel dari `:root` (tema light).
- Saat klik tombol, kita toggle `.theme-dark` di `<html>`, sehingga semua CSS variables yang didefinisikan di `.theme-dark` override nilai global dan UI berubah ke dark-ish.
- Semua komponen Bulma yang mengandalkan `--bulma-scheme-main`, `--bulma-body-background-color`, dan `--bulma-text` ikut berubah tanpa ubah HTML komponen.

### Langkah 2: Eksperimen cepat di halaman kamu

Setelah nyaman dengan `theme-demo.html`, coba langkah-langkah ini di project Bulma yang sudah ada:

1. Tambahkan blok CSS kecil di `<head>` satu file, misalnya di `dashboard.html`:

   ```html
   <style>
     :root {
       --bulma-primary-h: 171; /* hijau kebiruan */
       --bulma-primary-s: 100%;
       --bulma-primary-l: 41%;
     }
   </style>
   ```

2. Refresh dan lihat: semua `is-primary` di dashboard (navbar, button, tag, dsb.) berubah warna.

3. Kalau mau tema khusus hanya di dashboard, ganti `:root` dengan `.dashboard-theme` dan bungkus isi halaman dashboard kamu dengan `<div class="dashboard-theme">...</div>`.

---

Kalau sesi ini sudah masuk, berarti kamu:

- Tahu bahwa Bulma modern pakai CSS variables untuk semua komponen.
- Bisa override warna utama dan scheme background tanpa menyentuh Sass.
- Paham cara membuat dark mode sederhana dengan toggling class yang mengubah CSS variables.
