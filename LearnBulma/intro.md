# Pengantar Bulma CSS

Selamat datang di modul pembelajaran Bulma CSS!

Bulma adalah **framework CSS modern** yang dibangun sepenuhnya di atas **Flexbox**. Berbeda dengan framework lain yang mungkin masih mengandalkan float atau sistem grid tradisional, Bulma memanfaatkan Flexbox sebagai fondasi utama untuk semua komponen dan tata letaknya.

## Mengapa Memilih Bulma?

-   **Pure CSS**: Tidak memerlukan JavaScript untuk komponen dasar, membuat proyek Anda lebih ringan.
-   **Flexbox-based**: Menyediakan tata letak yang intuitif, powerful, dan mudah dipahami.
-   **Modular**: Anda dapat mengimpor hanya komponen yang dibutuhkan (jika menggunakan NPM), mengoptimalkan ukuran file CSS Anda.
-   **No JavaScript dependencies**: Bisa digunakan dengan vanilla JS atau framework JavaScript lainnya tanpa konflik.
-   **Modern**: Menggunakan fitur-fitur CSS3 terbaru seperti variabel dan Flexbox.
-   **Ringan**: Ukuran file yang relatif kecil (sekitar ~200KB setelah diminifikasi) untuk seluruh framework melalui CDN.

## Apa yang Akan Anda Pelajari?

Sepanjang seri tutorial ini, Anda akan dibekali dengan pemahaman teknis yang solid mengenai:

1.  **Konsep Inti**: Memahami secara mendalam tentang sistem grid Flexbox, sistem modifier Bulma, dan cara menciptakan desain yang responsif.
2.  **Layouting**: Mampu membuat berbagai tata letak, mulai dari halaman arahan (Hero), dasbor lengkap dengan Sidebar dan Navbar, hingga formulir yang kompleks.
3.  **Komponen Data**: Mahir dalam menggunakan komponen seperti Tabel (Table), Kartu (Card), Objek Media (Media Object), dan Level untuk menampilkan dan mengelola data.
4.  **Interaktivitas**: Menghidupkan antarmuka pengguna (UI) yang statis menggunakan Vanilla JavaScript, termasuk mengimplementasikan Modal Pop-up, sistem Tab, dan Dropdown.

## Instalasi Cepat (CDN)

Untuk memulai, Anda cukup menambahkan satu baris kode di bagian `<head>` file HTML Anda:

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bulma@0.9.4/css/bulma.min.css"
/>
```

Ini sudah termasuk semua komponen dasar seperti grid, form, button, navbar, dan lainnya. Anda tidak memerlukan build tools, npm, atau konfigurasi tambahan. Ini adalah cara tercepat dan paling efektif untuk memulai bagi pemula.

Mari kita mulai petualangan Anda dengan Bulma CSS!
